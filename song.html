Tune Finder

Song Recommender System

  • I Developed a content-based song recommendation system using machine learning and Streamlit. Utilized content-based filtering to recommend songs based on similarity of artist, genre, album name, and song name.
  • Engineered features from song metadata to compute similarity scores using cosine similarity.
  • Built an interactive and user-friendly web interface with Streamlit for real-time song recommendations, ensuring efficient performance by optimizing the machine learning model.
  • The project involved preprocessing and analyzing a dataset of songs, leveraging Python libraries such as pandas, numpy, and scikit-learn.
  • Source code :https://github.com/Shrutiii3/TuneFinder
